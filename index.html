Meus Projetos

Mineração de Dados Para Caracterizar Indivíduos Hipertensos com Doenças Cardiovasculares

Projeto que consistiu na criação de uma base de dados ideal para aplicações de Machine Learning, a base usou como referência a Pesquisa Nacional de Saúde 2019 disponibilizada pelo IBGE. O trabalho foi publicado no Congresso SBBD 2025 (Sociedade Brasileira de Banco de Dados) na Trilha Principal do Evento.

Gerenciamento de Leads

O projeto teve por objetivo criar uma automação capaz de lidar com as demandas preenchidas por leads via formulário, a automação capta as informações contrói um card dentro de uma lista correspondente à categoria da empresa que lida com esse segmento. Além disso, há a criação automática de um checklist de tarefas à se fazer para cumprir essa demanda. Por fim, utilizou-se agente para criar um e-mail de resposta ao cliente informando sobre o recebimento do projeto e requisitando uma data de reunião para discutir o projeto de forma mais detalhada para ser desenvolvido.

Análise Exploratória de Dados

Projeto que envolveu análise estatística minuciosa sobre o mercado de tecnologia na área de dados, seus diferentes cargos, salários, localizações e empresas. A técnica de análise exploratória de dados pode ser aplicada em qualquer contexto que há a disponibilização de dados.
